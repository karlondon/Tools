Gild3d — Platform Overview

Version: 1.1 Date: April 2026 Classification: Confidential Audience: Executive / Founders / Investors
01 Vision & Platform Purpose

Gild3d is a premium, privacy-first digital platform connecting discerning clients with vetted companion professionals — built on principles of discretion, security and luxury experience.

The platform addresses a significant gap in the marketplace: high-net-worth individuals seeking curated, trustworthy companion introductions currently rely on fragmented, insecure or low-quality digital channels. Gild3d provides an end-to-end managed experience — from discovery and booking through to secure payment — under a unified luxury brand.

"Gild3d is not a directory. It is a curated membership ecosystem where every interaction — discovery, booking, payment — is designed to the standard its clientele expects."

03 Key Platform Capabilities

Member Onboarding & Authentication

Email registration with OTP verification, JWT session management, role-based access. Secure password hashing (bcrypt). Rate-limited login protection.

Companion Profiles

Rich media profiles with photo galleries, private media vault, availability settings, location, rates and service descriptions. Admin-controlled publishing.

Booking System

Full booking lifecycle — request, payment, confirmation, cancellation. Supports in-call and out-call types. Hotel/location details captured securely. Minimum hour enforcement.

Cryptocurrency Payments

NOWPayments integration for privacy-preserving crypto transactions. HMAC-SHA512 webhook verification. Supports booking payments and membership tier upgrades.

Admin Dashboard

Admin Control Panel

Super Admin panel for companion management (add, edit, delete), member management, booking audit, message audit, platform analytics and revenue reporting.

Security Architecture

Helmet security headers, CORS protection, rate limiting at Nginx and Express layers, Cloudflare real-IP forwarding, file upload validation and cryptographic OTP generation.

Membership Tiers

Four-tier model (Free, Silver, Gold, Platinum) with crypto payment upgrade flow. Tier-gated feature access for exclusive content and companion interactions.

Messaging

Direct messaging system between members and companions. Sent/received message tracking with audit capability available to platform administrators.

VIP Exclusive Companion Tier

Admin-designated VIP companions visible and bookable only by PLATINUM members. Browse shows locked cards for lower tiers; profile pages display an upgrade wall. Admin can grant or revoke VIP status with one click.

Automated Platform Maintenance

Seven scheduled cron tasks running unattended: SSL certificate renewal, weekly server updates, daily PostgreSQL backups (14-day retention), container health monitoring with auto-recovery, disk space alerting, log rotation, and Docker cache cleanup. All failures trigger email alerts and site banners automatically.

05 Technology at a Glance

Frontend

Next.js 14 (App Router), React 18, TypeScript. Server-side rendering for SEO and performance. Responsive design.

Backend

Node.js + Express + TypeScript REST API. Prisma ORM with PostgreSQL 15. JWT authentication with secure HttpOnly cookies.

Infrastructure

Docker Compose on AWS Lightsail. Nginx reverse proxy with SSL termination. Cloudflare WAF + CDN layer. Auto-renewing Let's Encrypt certificates.

Security

OWASP Top 10 coverage. Helmet headers, rate limiting, input validation, file type enforcement, CORS protection, cryptographic token generation.

07 Strategic Value & Differentiators

Gild3d is positioned at the intersection of three macro trends: the normalisation of premium online service platforms, the growing adoption of cryptocurrency for privacy-sensitive transactions, and rising consumer expectations for luxury digital experiences.

The platform's key differentiators versus existing alternatives are:

Privacy by Design

Crypto-only payments, no card data stored, minimal data collection, secure messaging — built for clients who value absolute discretion.

Luxury Brand Positioning

Premium UX, curated companion profiles, tier-based exclusivity — differentiated from commodity directory services by quality and brand trust.

Admin-Controlled Quality

Every companion profile is admin-approved and published. No self-service listings. Quality control baked into the platform architecture.

Full-Stack Ownership

Client owns the complete codebase, infrastructure and data. No dependency on third-party SaaS platforms for core functionality.

Investment Readiness: The platform is production-ready and revenue-generating from day one of member onboarding. Phase 1 investment is in infrastructure and development; ongoing costs are primarily hosting and marketing. The business model is scalable with near-zero marginal cost per additional member.
